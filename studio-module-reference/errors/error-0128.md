# Error 0128  
 Number of conditional probabilities for categorical columns exceeds limit.  
  
## Resolution  
  
|Exception Messages|  
|------------------------|  
|Number of conditional probabilities for categorical columns exceeds limit.|  
|Number of conditional probabilities for categorical columns exceeds limit. Columns '{0}' and '{1}' are the problematic pair.|
